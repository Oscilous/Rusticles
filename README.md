# Rusticles
Simple sand/water particle physics simulation implemented in Rust

## Running the project

1. Clone the repo

```sh
git clone https://github.com/Oscilous/Rusticles.git
```

2. Change directory to the Rusticles project
   
```sh
cd Rusticles
```

3. Build the executable
```sh
cargo build
```
4. Run the executable
```sh
cargo run
```
